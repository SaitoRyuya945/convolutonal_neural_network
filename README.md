# 畳みこみニューラルネットワークを使おう

## 解説資料

- [page](https://saitoryuya945.github.io/convolutonal_neural_network/)

## 依存環境

- Python3.8.2
- PyQt5
- Numpy
- Matplotlib
- PIL
- pytest

## 実行方法

- CLIでの実行方法  
distフォルダで以下のようにライブラリをインストール

```bash
pip install dist/convolutonal_neural_network-1.0.tar.gz
```

CLIで以下を実行

```bash
convolutonal_neural_network
```
